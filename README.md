# ATM Interface 🏧

### 👨‍💻 Developer Details
* **Name:** Ishant Khare
* **Company:** CodSoft

### 🎯 Objective
To design an Object-Oriented console application that simulates a real-world ATM machine, allowing users to withdraw, deposit, and check their balance.

### 🛠️ Tools Used
* Java
* Object-Oriented Design (Classes and Objects)

### 📝 Steps Performed
1. Created a `BankAccount` class to securely hold and update the user's balance.
2. Created an `ATM` class providing a menu-driven interface for transaction options.
3. Implemented logic to ensure successful deposits and prevent overdrawing during withdrawals.

### 💡 Outcome
A robust financial simulation that safely manages state and separates logic using core OOP principles.
